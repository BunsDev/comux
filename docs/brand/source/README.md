# comux Brand Sources

This directory defines the source layout for comux brand assets used by the documentation site.

- `comux-wordmark.svg` is the master vector wordmark.
- `comux-mark.svg` is the compact mark.
- `comux-hero.svg` is the still composition rendered for public docs.
- `comux-og.svg` is the social card source.
- `comux-motion-scene.mjs` is reserved for future motion rendering.
